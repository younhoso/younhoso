# 개발자 웹 이력서 프로젝트 로드맵

## 프로젝트 개요

개인 브랜딩과 취업/이직을 위한 개발자 웹 이력서 사이트 구축

---

## 기술 스택 (MVP)

> 📁 참조: `config/mvp-stack.yaml`

| 구분      | 기술                             |
| --------- | -------------------------------- |
| Frontend  | HTML5 + CSS3 + Vanilla JS (ES6+) |
| AI 서비스 | Google Gemini 1.5 Flash          |
| 배포      | Vercel / Netlify                 |

---
