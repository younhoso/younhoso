# 디자인 시스템

> 개발자 웹 이력서 프로젝트의 디자인 가이드

---

## 레이아웃

- **모바일 우선**: 기본 최대 너비 480px
- **데스크톱**: 481px 이상에서 카드형 레이아웃
- **터치 영역**: 최소 44px

---

## 색상

| 용도 | 색상코드 |
|------|----------|
| 배경 | `#f4f5f7` |
| 텍스트 | `#191f28` |
| 보조 텍스트 | `#8b95a1` |
| 액센트 | `#3182f6` |

---

## 폰트

```css
font-family: -apple-system, BlinkMacSystemFont, 'Apple SD Gothic Neo', sans-serif;
```

- 시스템 폰트 스택 사용
- 외부 폰트 로드 없음 (성능 최적화)

---

## 아이콘

- 인라인 SVG 사용
- 외부 아이콘 라이브러리 없음

---

## 주요 기능

### 프로필 공유
- Web Share API 지원 시: 네이티브 공유
- 미지원 시: 클립보드 복사 폴백

### 토스트 알림
- DOM에 동적 생성
- 자동 사라짐 (3초)

### 이미지 폴백
- 프로필 이미지 로드 실패 시 SVG 플레이스홀더 표시

### 섹션별 애니메이션
- fadeIn 애니메이션으로 순차 등장 효과
