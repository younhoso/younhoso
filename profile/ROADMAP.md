## Phase 1: MVP (현재 단계)

### 목표
최소 기능으로 빠르게 검증

### 기술 스택
- HTML5 + CSS3 + Vanilla JS (ES6+)
- 배포: Vercel / Netlify

### 핵심 기능
- [ ] 프로필 페이지 기본 구조
- [ ] 모바일 반응형 레이아웃
- [ ] 프로필 공유 기능 (Web Share API)
- [ ] 토스트 알림
- [ ] 이미지 폴백 처리

### 완료 기준
- 모바일에서 정상 동작
- 공유 기능 테스트 완료
- 기본 SEO 메타 태그 적용
