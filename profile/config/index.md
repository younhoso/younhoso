| 파일            | 설명                                |

## 레이아웃

- **모바일 우선**: 기본 최대 너비 480px
- **데스크톱**: 481px 이상에서 카드형 레이아웃
- **터치 영역**: 최소 44px

---

## 아이콘

- 인라인 SVG 사용
- 외부 아이콘 라이브러리 없음

---

## 주요 기능

### 프로필 공유

- Web Share API 지원 시: 네이티브 공유
- 미지원 시: 클립보드 복사 폴백

### 토스트 알림

- DOM에 동적 생성
- 자동 사라짐 (3초)

### 이미지 폴백

- 프로필 이미지 로드 실패 시 SVG 플레이스홀더 표시

### 섹션별 애니메이션

- fadeIn 애니메이션으로 순차 등장 효과
