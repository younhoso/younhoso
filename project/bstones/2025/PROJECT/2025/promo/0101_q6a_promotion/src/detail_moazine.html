‘모아진’ 국내·해외 디지털 매거진
3개월 무제한 구독권

  • 이 혜택은 이벤트 기간 Galaxy Z Fold Special Edition 구매자에게만 적용되는 혜택으로, 구매한 휴대폰으로 가입 및 접속했을 경우 적용됩니다.
  • 무료 구독 신청 시 유효한 결제 방법을 등록해야 하며, 무료 구독 기간이 끝날 때 까지는 요금이 청구되지 않습니다.
  • 3개월 무료 구독 기간에는 위약금이나 약정 없이 언제든 해지가 가능하며, 해지 이후 남은 구독 기간까지 이용이 가능합니다.
  • 모아진을 이미 구독하고 계신 고객은 구독권을 사용하시면, 기존 구독 기간에 무료 기간이 합산이 되어 제공이 됩니다.
  • 이벤트는 구매 대수만큼 적용이 가능하며, 하나의 아이디에 중복 적용 가능합니다.
  • 해당 이벤트 프로모션 구독권 사용 후 다른 무료 구독 프로모션 사용이 불가합니다.
  • 관련 문의 : 모아진 고객센터 (1660-0838, 월~금 09~18시, 점심 시간 12~13시)
