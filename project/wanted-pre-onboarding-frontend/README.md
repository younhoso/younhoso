## 프로젝트의 실행 방법
```
$ npm install
$ npm start

or

$ yarn
$ yarn start
```
- 환경 변수 설정 필요(.env)

## packages 설정
- react-router-dom: 6.4.2
- axios: 1.1.2
- styled-components: 5.3.6
- react-icons: 4.6.0
