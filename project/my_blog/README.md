## 프로젝트 실행방법
실행할 때 반드시 다음 파일을 최상단에 생성후 실행 해주셔야 합니다.
```
// .env.local
NOTION_TOKEN='TOKEN 값'
NOTION_DATABASE_ID='DATABASE_ID 값'
```
```
npm install // 설치
npm run dev // 실행
```
