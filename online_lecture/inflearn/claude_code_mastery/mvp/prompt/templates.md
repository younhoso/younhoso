## 1. 작업 지시문 템플릿

```markdown
# [작업명]

> 작업 목적을 한 줄로 설명

## 목표

- [ ] 목표 1
- [ ] 목표 2

## 제약 조건

- MVP 수준 유지
- 기존 코드 스타일 준수

## 참고 파일

- `config/mvp-stack.yaml`
- `config/design-system.md`

## 완료 기준

- [ ] 기능 정상 동작
- [ ] 반응형 확인
```

## 2. 기능 요청 템플릿

```markdown

## 개요

**사용자 스토리**: [사용자]로서, [목표]를 위해, [기능]이 필요합니다.

## 요구사항

### 필수

1. 요구사항 1

### 권장

1. 요구사항 2

## UI/UX

- 위치:
- 인터랙션:
- 반응형:

## 우선순위

- [ ] P0 - MVP 필수
- [ ] P1 - MVP 권장
- [ ] P2 - 확장 단계
```
